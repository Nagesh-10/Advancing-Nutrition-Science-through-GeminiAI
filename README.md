
# NutriGen - Gemini AI Nutrition App

## Features
- Dynamic Nutritional Insights
- AI-Based 7-Day Meal Plan Generator
- Built using Flask + Google Gemini API

## Setup Instructions

1. Install dependencies:
   pip install -r requirements.txt

2. Add your Gemini API key inside app.py

3. Run the app:
   python app.py

4. Open browser:
   http://127.0.0.1:5000
